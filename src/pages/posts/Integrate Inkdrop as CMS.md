## yarn start
```zsh
yarn start #start localhost:3000
```
<br>

## LIVE-IMPORT
```zsh
yarn run live-import # link inkdrop db
```
<br>

## `--enable-logging` parameter.
```zsh
/Applications/Inkdrop.app/Contents/MacOS/Inkdrop --enable-logging
```
<br>
